**Guia de Inteligência Financeira: Do Comportamento à Liberdade**
--------------------

Bem-vindo ao repositório do **Guia de Inteligência Financeira**. Este projeto foi desenvolvido com o auxílio da inteligência artificial (NotebookLM), utilizando como base os ensinamentos de autoridades no tema, como **Robert Kiyosaki** (_Pai Rico, Pai Pobre_), **Morgan Housel** (_A Psicologia Financeira_) e **James Jani**.<br>
O objetivo central deste guia é transformar a percepção comum sobre o dinheiro, movendo o foco de "trabalhar arduamente pelo capital" para "fazer o dinheiro trabalhar arduamente para você".
<br><br>**📌 Sobre o Guia de Inteligência Financeira**
<br><br>O guia contido neste repositório não é apenas um manual de fórmulas matemáticas, mas um roteiro de mudança de mentalidade (mindset). Ele explora a premissa de que o sucesso financeiro depende menos da inteligência técnica e muito mais do comportamento individual.
<br><br> **Principais Pilares Abordados:**

1.  **Alfabetização Financeira:** A base necessária para distinguir a história real que os números contam, permitindo identificar corretamente o que são **Ativos** (que põem dinheiro no seu bolso) e **Passivos** (que tiram dinheiro dele).
    
2.  **Psicologia do Dinheiro:** Uma análise sobre como nossas experiências passadas moldam nossa visão de risco e a importância de diferenciar **Riqueza** (o que você vê) de **Fortuna** (os ativos que você não vê e que compram sua liberdade).
    
3.  **Mecanismos de Construção de Riqueza:** O uso do **Tempo** como o ingrediente mais crítico nos **Juros Compostos** e a busca por soluções **Escaláveis** que resolvem problemas para o mercado em larga escala.
    
4.  **Estratégias de Sobrevivência:** A aplicação da **Margem de Segurança** para sobreviver aos imprevistos e permanecer no "jogo" tempo suficiente para que a composição exponencial atue.

<br>**🚀 Como utilizar este material**

1.  **Comece pelo Guia Principal:** Leia o Guia\_Inteligencia\_financeira.md para entender a progressão lógica da construção de riqueza.
    
2.  **Consulte o Glossário:** Sempre que encontrar um conceito novo, o Glossario.md fornecerá definições claras, exemplos práticos e relações entre os termos.
    
3.  **Estude as Fontes:** Assista aos vídeos listados em Fontes.md para aprofundar-se nas nuances explicadas por cada autor.
    
4.  **Pratique com os Prompts:** Utilize o arquivo Prompts.md para realizar suas próprias revisões ou para aprofundar temas específicos em ferramentas de IA.
